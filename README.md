# Foxhole

## The Crypto Twitter Intelligence Platform

Foxhole helps traders avoid scams, instantly grasp meme coin narratives, and discover the next 1000x gem before it trends.

### Requirements

- Node.js 20+ and npm

### Getting started

Run the following command on your local environment:

```shell
npm install
```

For your information, all dependencies are updated every month.

Then, you can run the project locally in development mode with live reload by executing:

```shell
npm run dev
```

Open http://localhost:3000 with your favorite browser to see the project.

### Testing

All unit tests are located alongside the source code in the same directory, making them easier to find. The project uses Vitest and React Testing Library for unit testing. You can run the tests with the following command:

```shell
npm run test
```
